Etiquetas de html

  • Los elementos cabecera. Van del 1 al 6.
  • Los párrafos. Se indican con la etiqueta <p>
  • La etiqueta <hr> permite crear una línea horizontal.
  • Los enlaces se crean con la etiqueta a que requiere el atributo href.
  • Las listas:
    • Las ol son ordenadas y muestran números.
    • las ul no están ordenadas y muestran puntos.
  • Las tablas permiten estructurar información. Esto se puede ver en la sección de aerogeneradores.
  • Las etiquetas input y select permiten crear formularios para que el usuario introduzca información.
